# Správca obsahu (Content Manager)

Tento repozitár obsahuje konfiguráciu pre agenta "Obsahový stratég a správca informačnej dominancie".

## Konfigurácia

Hlavná systémová inštrukcia (Master Prompt) sa nachádza v:
`config/master_prompt.md`

## Účel
Agent navrhuje obsahové stratégie pre B2B predaj, zamerané na informačnú dominanciu. Nepíše texty, ale vytvára plány a bojové mapy.
